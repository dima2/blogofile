Ubuntu Faq:

  • Сразу после установки Ubuntu следует выполнить: [code]sudo apt-get install ubuntu-restricted-extras sudo apt-get install preload[/code]Первая комманда установит все необходимое для нормальной работы - кодеки, аудио-видео, поддержку архивов, а вторая ускорит запуск программ.
  • Убить пищалку или выключить наконец надоедливый Pc-Speaker: [code]sudo gedit /etc/modprobe.d/blacklist[/code] добавить в конец строку "blacklist pcspkr"
  • CapsLock выключается в Ubuntu: "System > Preferences > Keyboard > Layout > Layout Options > Ctrl key position > Make CapsLock Additional Ctrl"
  • Слишком частая проверка дисков. (каждые 30 запусков, иногда раздражает если включать/выключать компьютер по нескольку раз в день) [code]sudo tune2fs -c 0 -i 1m /dev/sda2 sudo tune2fs -c 0 -i 1m /dev/sda5 ... и так для всех дисков[/code]Это назначит проверку раз в месяц.
  • Как подключить клавиатуру от Apple или любую другую USB клавиатуру в Ubuntu: здесь.
  • Ubuntu: какой софт устанавливать.
Теперь про проблемы с софтом:
  • Firefox прячет показывает меню при нажтии alt (сильно раздражает при смене раскладки) "about:config -> compact.menu.altToggle"
  • Firefox не хочет устанавливать аддоны: (несекьюрно) "about:config -> NEW KEY bool extensions.checkUpdateSecurity"
  • Firefox 3 жизненно необходимый плагин Tab Mix Plus находится: http://tmp.garyr.net/dev-builds/
  • Не работает Java в Firefox 3: [code]sudo apt-get remove icedtea-gcjwebplugin[/code]
  • Utorrent под Ubuntu сильно тормозит: "Utorrent->Options->Prefences->Connection->" Убрать птичек "NAT-PMP" и "Windows Firewall""
  • Ubuntu создать архив с паролем: Создать пустой архив, затем: "Edit->Password->Set Password" и только потом добавлять файлы.
  • Ubuntu установить и настроить Photoshop CS2: 1. Установить свежую версию Wine 2.[code]ln -s /dev/sda ~/.wine/dosdevices/c:: sudo chmod 666 /dev/sda[/code]3.Нормально установить и активировать  Photoshop CS2 3.[code]sudo chmod 660 /dev/sda[/code]
  • Посмотреть сколько свободной оперативки: [code]free -m[/code]
  • После установки Virtual Box следует выполнить [code]sudo gpasswd -a USER vboxusers[/code] после чего LogOut, и зайти заново.
  • Установить последнюю версию Wine: [code]wget -q http://wine.budgetdedicated.com/apt/387EE263.gpg -O- | sudo apt-key add - sudo wget http://wine.budgetdedicated.com/apt/sources.list.d/hardy.list -O /etc/apt/sources.list.d/winehq.list sudo apt-get update sudo apt-get install wine [/code]
  • Ubuntu не открывается Rar, не играет видео/музыка: [code]sudo apt-get install ubuntu-restricted-extras[/code]
  • Настройки клавиатуры не сохраняются: [code]sudo gedit /etc/X11/xorg.conf[/code] и изменяем все вручную. У меня (выключенный CapsLock, переключение по Alt+Shift) выглядит вот так: [code]Section "InputDevice" Identifier "Generic Keyboard" Driver "kbd" Option "CoreKeyboard" Option "XkbRules" "xorg" Option "XkbModel" "pc105" Option "XkbLayout" "us,ru" Option "XkbOptions" "grp:alt_shift_toggle, ctrl:nocaps" EndSection[/code]
  • При проигрывании видео слишком яркие/неправильные цвета: В настройках видео плеера измените output c "xv" на "x11".
