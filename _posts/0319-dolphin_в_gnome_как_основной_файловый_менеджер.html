Я ничего не имею против Gnome, как среды рабочего стола. Но его дефолтный файловый менеджер Nautilus, меня как говориться "просто бесит". Потому что малофункционален. Я тоже поддерживаю принцип "ничего лишнего", но об удобстве в конце концов тоже нужно подумать! Что-бы не быть голословным, вот преимущества KDE'шного Dolphin'а против Nautilus'а:

  • Возможность включить двухпанельный интерфейс. Очень удобно на маленьких экранах типа моего Asus EEE 1000H.
  • Для определенных папок, удобно включать (List View) и предпросмотр (Preview) . А не сразу для всех как у Nautilus.
  • Приятный внешний вид.
  • Подробная информация о файлах и папках.
  • Архивы отображаются как папки.
  • Удобная фишка Filter, позволяющяя показывать/выделять только нужные файлы.
  • И наконец, кликабельная адресная строка.
  • + еще множество маленьких удобств. Смотрите скриншот.

Можно было бы просто поставить KDE, мне в принципе все равно. Но в Ubuntu EEE из коробки шел настроенный гном, с парой незаменимых виджетов. Проще было прикрутить к Gnom'у человеческий FM, чем настраивать Кеды с нуля. Что я собственно, и сделал.

UPD: Для Ubuntu 9.10 и выше все просто: 

[code lang='bash']sudo apt-get install dolphin ark[/code]
что-бы устаноился только dolphin, без всего KDE (+ark для разархивирования из меню).

Что-бы полностью заменить Nautilus, создайте исполняемый файл ~/bin/nautilus с содержимым:

Код:
[code lang='bash']#!/bin/bash
exec dolphin "@"
exit 0[/code]
Для Ubuntu 8.04:

[caption id="attachment_320" align="alignleft" width="150" caption="dolphin kde3"][/caption]

  1. [code]sudo apt-get install dolphin[/code] Если установить Dolphin чepез apt-get со всеми зависимостями, нормально работать ничего не будет :(. А именно - сеть (remote:/ ), мусорник (trash:/ ) и прочие системные папки требующие io slave. Что-бы заставить их отображаться нужна еще пара пакетов: [code lang='bash']sudo apt-get install kdebase-bin kdebase-kio-plugins[/code]
  2. Переходим к проблеме номер 2 - Dolphin в Gnome'вской среде будет выглядеть как неродной. Поэтому устанавливаем и запускаем kcontrol: [code lang='bash']sudo apt-get install kcontrol && kcontrol[/code] Идем в "Apperance & Themes" где выбираем  гномовские цвета и иконки. Кстати, это подправит все KDE'шные приложения.
  3. Когда все работает, делаем что-бы папки открывались Dolphin'ом по умолчанию: [code lang='bash']sudo gedit /usr/share/applications/nautilus-folder-handler.desktop[/code] и заменяем "nautilus --no-desktop %U" на "d3lphin". Что-бы быстрее открывался при запуске ставим preload: [code]sudo apt-get install preload[/code]
  4. Upd: Что-бы при вызове Nautilus внешними программами (например DropBox) вместо Nautilus открывался Dolphin выполните комманду:[code lang='bash']mkdir -p ~/bin && touch ~/bin/nautilus && chmod +x ~/bin/nautilus && gedit ~/bin/nautilus &amp;amp;[/code] в открывшейся файл скопируйте [code lang='bash']#!/bin/bash exec dolphin "$@" exit 0[/code]
  5. Для удобного umount'а флешек, добавляем на панель Gnom'а "Drawer", а в него "Disk mounter"
  6. Ну и наконец, добавим в меню Actions немного функциональности  - поиск, архиваторы, скрипт для переименования множества файлов (KRename), удаление файлов минуя мусорник, редактирование любого файла Gedit'ом: [code lang='bash']sudo apt-get install kfind krename ark[/code]  После установки, скачайте у меня готовые расширения для меню ([download#23#nohits]) и раззипуйте в папку "~/.kde/share/apps/d3lphin/servicemenus".
Пользуемся удобным и функциональным fm ;).
