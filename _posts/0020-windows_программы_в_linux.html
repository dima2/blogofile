Конечно почти у всех самых популярных Windows (.exe) приложений есть Linux аналоги. (Про набор софта для Ubuntu читайте здесь). Но бывает, что какая нибудь малоизвестная, однако очень нужная програмка есть только под виндой. Или, например такие программы как Strong DC, Utorrent, Total Commander, ИМХО заменить невозможно, т.к они просто лучшие в своем роде.

Кстати, под wine, можно запустить даже Internet Explorer! Подробнее.

На всякий случай, кусок банальщины.

Всем известно, что для запуска Windows приложений под Linux, нужен wine. Добывается он через консоль:

[code]sudo apt-get install wine[/code]

"Показать" Windows проге разделы жесткого диска, можно через

[code]winecfg[/code]

Через консоль Windows программу можно легко запустить:

[code]wine путь/прога.exe[/code]

В папку "~/.wine/drive_c/windows/fonts" хорошо бы скопировать откуда-нибудь Windows'овские шрифты. Запускать софт с root привелегиями (возможностью изменять системные файлы) не рекомендуется. Кто знает, что она может натворить? На всякий случай, в консоли, это делается через "sudo wine/прога". Правильная команда в Launcher'e:

[code]gksudo env WINEPREFIX="/home/имя_пользователя/.wine" wine "путь/прога.exe"[/code]

Еще раз повторюсь, что этого делать не стоит. Правильным решением будет поставить права 777 на папку в которой собираетесь работать виндовым приложением.

Если в wine софт не запустился, есть два варианта. Первый - искать в интернете решение. Иногда действительно можно найти неочевидный способ, с regedit похимичить, например. Но это сложно, и не всегда себя оправдывает затраченные усилия.

А второй способ - запустить виртуально Windows. Путь наименьшего сопротивления, так сказать. Ресурсов системных это конечно сожрет больше (еще бы - две операционки одновременно), но нужная программа запуститься гарантированно. И ни каких проблем со шрифтами как у wine. Есть очень быстрая, бесплатная и простая программа для виртуализации - VirtualBox. С такими вещами как Webmoney и Photoshop, я работаю именно в ней.

Ну а для игр есть Cedega. Хотя для них лучше всего нормальную Windows поставить. Lite версию какую-нибудь. Чисто что-бы играться.
