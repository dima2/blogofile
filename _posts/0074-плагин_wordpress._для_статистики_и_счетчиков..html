Написал, как мне кажеться, полезный плагин для Wordpress. Позволяет видеть статистику Liveinternet.ru, (или любого другого подобного сервиса) прямо в админке Wordpress'а. Плюс возможность добавить код счетчика (щотчега) сразу на все страницы. Проще говоря, показывает в Dashboard'е выбранный сайт (во frame). Так что использовать плагин можно в разных целях ;) .

Установка и настройка для Liveinternet.ru.

  1. Загрузите grafs_frame.php в папку /wp-content/plugins/.
  2. Активируйте Graf's Frame в меню плагинов WordPress.
  3. Зайдите Options -> Graf 's Frame.
  4. В поле Name напишите название для таблички в Dashboard'е
  5. В Statistics adress скопируйте адрес сайта который вы хотите там видеть. Например cо статистикой Liveinternet.ru: "http://www.liveinternet.ru/stat/thexnews.com/index.html"
  6. В Counter code можно вставить код счетчика.
