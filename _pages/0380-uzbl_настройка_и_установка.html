Установка для Ubuntu:

Готовые пакеты из репозитория устанавливать не советую так как они сильно устарели. Свежую версию нужно компилировать из исходников.

  1. Добавляем репозитории WebKit в /etc/apt/sources.list [code lang='bash']deb http://ppa.launchpad.net/webkit-team/ppa/ubuntu intrepid main deb-src http://ppa.launchpad.net/webkit-team/ppa/ubuntu intrepid main[/code]
  2. Устанавливаем зависимости и GIT:[code lang='bash']sudo apt-get update && sudo apt-get install git git-core libwebkit-dev[/code]
  3. Скачиваем и компилируем свежую версию[code]git clone git://github.com/Dieterbe/uzbl.git cd uzbl make sudo make install[/code]
Следуя Философии сразу работать ничего не будет. Нужны правильные конфиги. Потом обязательно нужно будет настроить все по своему, а пока скопируем из примеров: [code lang='bash']cp -r /usr/share/uzbl/examples/data/s ~/.config cp /usr/share/uzbl/examples/config/uzbl/config ~/.config/uzbl/[/code]

Теперь можно запускать браузер коммандой uzbl или скриптом ~/.config/uzbl/scripts/uzbl_tabbed.py (с табами!)

Азы управления

[i] переключает в режим ввода теста (в некоторых версиях [I] c шифтом). То есть перед тем как написать что-либо в форму, нажимаем i. [Esc] переключает в режим ввода команды. Вот некоторые из них:

  • Чтобы перейти по ссылке: o адрес [Enter]
  • Чтобы найти что-либо на странице /чтоищем
  • Назад - b
  • Paste'n'go - p
  • открыть новый таб (если запущен uzbl_tabbed.py) gn
  • следующий таб gt
  • закрыть таб gC
Остальное можно посметреть и перенастроить под себя в файле "~/.config/uzbl/config"

Некоторые примеры настройки

Настраивается uzbl, естественно редактированием текстового файла "~/.config/uzbl/config"

Неудобство с "o" перед ссылкой компенсируется возможностью сделать шорткаты для часто посещаемых сайтов. Например, что-бы при наборе "d3" открывался dirty.ru: [code lang='bash']bind    d3        = uri http://dirty.ru[/code]

Удобно так-же добавить шорткаты для поисковиков. Поиск "gi чтоищем" по картинкам Google; "gs чтоищем" поиск на открытом сейчас сайте. [code lang='bash']bind gi _  = uri http://images.google.com/images?q=%s bind gs _   = sh "echo uri http://www.google.com/search?q=%s site:`echo $6` &amp;gt; $4"[/code]
